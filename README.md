# YouTubeArt  

## Описание  

:pencil2: -  Приложение может скачивать видео по ссылке из Youtube.  
:pencil2: -  Вы так же можете выбрать место, куда нужно сохранить скаченное видео.  

## Используемые библиотеки  

:pushpin: - PyQt5    
:pushpin: -  youtube_dl   
:pushpin: -  sys
